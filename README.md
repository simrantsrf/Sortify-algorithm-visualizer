VIEW PROJECT : https://sortify-algoritm-visualizer.netlify.app/

Sortify is an interactive web-based algorithm visualizer that helps users understand how sorting algorithms work through real-time
animations. It provides a clean and intuitive interface to explore different sorting techniques step by step.

Features :-
Visualizes popular sorting algorithms (Bubble, Selection, Insertion, etc.)
Real-time animation of comparisons and swaps
Responsive and user-friendly UI
Generates random datasets for practice

Advantages :-
Makes complex algorithms easy to understand visually
Great for students preparing for DSA and coding interviews
Enhances learning through interaction rather than theory
Helps in comparing efficiency of different algorithms
